# 珍珠奶茶

> 自制 Q 弹珍珠奶茶，黑糖珍珠软糯有嚼劲，茶香奶香完美融合，比外面买的更健康

## 基本信息

| 项目 | 内容 |
|------|------|
| 分类 | 饮品 |
| 口味 | 甜香顺滑 |
| 难度 | 简单 |
| 用时 | 约 45 分钟 |
| 份量 | 2 杯 |

## 食材

| 食材 | 用量 | 备注 |
|------|------|------|
| 木薯淀粉 | 100g | 必须是木薯淀粉，不可替代 |
| 红茶包 | 3 包（约 6g） | 推荐锡兰红茶或阿萨姆红茶 |
| 纯牛奶 | 300ml | 全脂牛奶口感更好 |
| 热水 | 200ml | 泡茶用 |

## 调料

| 调料 | 用量 | 备注 |
|------|------|------|
| 红糖/黑糖 | 50g | 煮珍珠用 |
| 细砂糖 | 20g | 奶茶调味，可根据口味增减 |
| 清水 | 70ml | 和面团用 |

## 用具

- 小锅（煮珍珠用）
- 杯子或量杯
- 砧板
- 粗吸管

## 做法

### 1. 准备工作

**制作珍珠：**
- 50g 红糖加 70ml 清水，小火煮至红糖完全融化并沸腾
- 趁热倒入 100g 木薯淀粉中（分次加入），用筷子搅拌成棉絮状
- 稍晾至不烫手，揉成光滑面团（面团偏硬是正常的）
- 搓成细长条，切成约 1cm 小粒，搓圆
- 撒少许木薯淀粉防粘

### 2. 烹饪步骤

**煮珍珠：**
- 大锅烧开足量的水（水量是珍珠的 10 倍以上），水沸后放入珍珠
- 大火煮 15 分钟，期间轻轻搅拌防粘底
- 关火盖盖焖 15 分钟
- 捞出珍珠过冰水，沥干后泡在少许红糖水中（15g 红糖 + 30ml 热水）保持甜味和光泽

**泡茶：**
- 200ml 沸水冲泡 3 个红茶包，盖盖焖 5 分钟至茶汤浓郁深红
- 取出茶包，加入细砂糖搅拌至融化
- 放凉至室温

**组装：**
- 杯中先放入 2-3 大勺煮好的珍珠
- 加入冰块至杯子半满
- 倒入红茶至杯子七分满
- 最后倒入牛奶，搅拌均匀即可

### 3. 装盘

- 用粗吸管饮用，吸管要足够粗才能吸起珍珠
- 可在杯口挂一片薄荷叶装饰

## 技巧与窍门

- 木薯淀粉不可用其他淀粉替代，只有木薯淀粉才能做出 Q 弹口感
- 红糖水一定要煮沸后再加入淀粉，热液体才能让淀粉糊化产生黏性
- 珍珠大小要均匀，否则煮出来软硬不一
- 煮珍珠水一定要足量且先烧开，冷水下锅珍珠会散
- 煮好的珍珠泡在糖水中可以保持 Q 弹 2-3 小时，时间长了会变硬
- 茶汤一定要泡浓一些，加了牛奶和冰块后茶味会被稀释
- 想要奶味更浓可以用 1:1 的茶奶比例

## 变体

- **黑糖脏脏奶茶**：用黑糖浆挂壁（黑糖 + 少量水熬至浓稠），沿杯壁淋下再倒入奶茶
- **椰奶珍珠茶**：牛奶换成椰浆，增加热带风味
- **鲜芋珍珠奶茶**：加入煮熟的芋头丁，双重 Q 弹口感
