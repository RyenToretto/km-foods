# 意大利肉酱面

> 经典博洛尼亚肉酱意面，肉酱浓郁醇厚，面条弹牙有嚼劲

## 基本信息

| 项目 | 内容 |
|------|------|
| 分类 | 西餐 |
| 口味 | 咸鲜微酸 |
| 难度 | 中等 |
| 用时 | 约 60 分钟 |
| 份量 | 2-3 人份 |

## 食材

| 食材 | 用量 | 备注 |
|------|------|------|
| 意大利面 | 200g | 直径 1.6-1.8mm 的 Spaghetti |
| 牛肉馅 | 250g | 七分瘦三分肥最佳，也可牛猪混合 |
| 番茄 | 3 个（约 400g） | 成熟的大番茄，或用 400g 罐装番茄块 |
| 洋葱 | 半个（约 100g） | 切小丁 |
| 胡萝卜 | 半根（约 60g） | 切小丁 |
| 西芹 | 1 根（约 50g） | 切小丁 |
| 蒜 | 3 瓣（约 10g） | 切末 |

## 调料

| 调料 | 用量 | 备注 |
|------|------|------|
| 番茄酱 | 2 汤匙（约 30g） | 浓缩番茄膏更佳 |
| 橄榄油 | 3 汤匙（约 45ml） | — |
| 红酒 | 100ml | 干红，可用料酒替代 |
| 盐 | 1 茶匙（约 6g） | 炒酱用 |
| 黑胡椒碎 | 1/2 茶匙（约 2g） | 现磨最香 |
| 干牛至（Oregano） | 1 茶匙（约 2g） | 意式料理灵魂香料 |
| 干罗勒（Basil） | 1 茶匙（约 2g） | — |
| 月桂叶 | 1 片 | — |
| 白糖 | 1 茶匙（约 5g） | 平衡番茄酸味 |
| 帕玛森芝士粉 | 20g | 出锅后撒，可选 |
| 煮面水用盐 | 15g | 每升水加 10g 盐 |

## 用具

- 深汤锅（煮面用）
- 平底炖锅或深煎锅（做肉酱用）

## 做法

### 1. 准备工作

- 番茄顶部划十字，开水烫 30 秒后捞出过冷水，去皮切小丁
- 洋葱、胡萝卜、西芹分别切成 5mm 小丁（意式经典 Soffritto 底料）
- 蒜切末

### 2. 烹饪步骤

- **炒肉酱**：锅中加入 2 汤匙橄榄油，**中大火**将牛肉馅炒散变色（不要急着翻动，让底部焦化产生风味），约 5 分钟，盛出备用
- 锅中再加 1 汤匙橄榄油，**中小火**炒洋葱丁至透明（约 3 分钟），加入胡萝卜丁和西芹丁继续炒 3 分钟
- 加入蒜末炒出香味（约 30 秒）
- 加入番茄酱翻炒 1 分钟，使其颜色变深
- 倒入红酒，**大火煮至酒精挥发**（约 2 分钟），液体减半
- 加入番茄丁、炒好的肉馅、干牛至、干罗勒、月桂叶、白糖
- 翻炒均匀后**转小火，盖盖焖煮 30 分钟**，期间偶尔搅拌防粘底
- 加盐和黑胡椒调味，取出月桂叶

- **煮面**：深锅烧一大锅水（至少 2L），水开后加盐 15g
- 下入意大利面，按包装标注时间**减 1 分钟**捞出（保持 al dente 口感）
- 捞面前舀出 1 小碗煮面水备用

- **合酱**：将煮好的面条倒入肉酱锅中，加 2-3 汤匙煮面水
- **中火翻拌 1-2 分钟**，让面条充分吸收酱汁（面条会继续吸水变熟达到完美口感）

### 3. 装盘

- 用食物夹将面条卷起高高地盛入盘中
- 顶部放上一勺肉酱
- 撒上帕玛森芝士粉和少许新鲜罗勒叶

## 技巧与窍门

- 煮面水要加足盐（尝起来像淡海水），这是面条入味的唯一机会
- 面条一定要 al dente（有芯），捞出后会在酱中继续变软
- 煮面水含淀粉，是天然的增稠剂，能让酱汁更好地挂在面条上
- 肉馅炒的时候不要急着搅碎，让底部焦化（梅拉德反应）能带来更丰富的风味
- 好的肉酱需要时间，小火慢煮让各种食材的味道充分融合

## 变体

- **奶油肉酱面**：最后加入 50ml 淡奶油拌匀，口感更柔和
- **茄子肉酱面**：加入切丁的茄子 1 个，和蔬菜丁一起炒
- **素版肉酱面**：用 200g 碎豆腐或植物肉代替牛肉馅
- **千层面（Lasagna）**：肉酱搭配白酱和千层面皮交替叠放，烤箱 180°C 烤 30 分钟
