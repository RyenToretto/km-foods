# 凯撒沙拉

> 经典西式开胃沙拉，酱汁浓郁咸香，面包丁酥脆，层次丰富

## 基本信息

| 项目 | 内容 |
|------|------|
| 分类 | 西餐 |
| 口味 | 咸鲜微酸 |
| 难度 | 简单 |
| 用时 | 约 25 分钟 |
| 份量 | 2 人份 |

## 用具

- 烤箱或平底锅（烤面包丁）
- 搅拌碗
- 手动打蛋器

## 做法

### 1. 准备工作

- 罗马生菜洗净，甩干水分（水分太多酱汁挂不住），撕成 5cm 大小的片
- 吐司去边切成 1.5cm 方丁，拌上橄榄油、蒜粉和少许盐
- 面包丁放入烤箱 **180°C 烤 8-10 分钟**至金黄酥脆（或平底锅小火煎至酥脆），取出放凉
- 鸡胸肉两面撒盐和黑胡椒

### 2. 烹饪步骤

- **煎鸡胸肉**：平底锅中火加少许油，鸡胸肉**每面煎 5-6 分钟**（厚度 2cm 的情况下），至中心温度 74°C 或切开无粉红色。煎好后静置 5 分钟再斜切成薄片
- **制作凯撒酱**：碗中放入蛋黄、蒜泥、鳀鱼碎、芥末酱、柠檬汁，用打蛋器搅匀
- **缓慢加入橄榄油**，一边倒一边快速搅打（类似做蛋黄酱），直到酱汁乳化变浓稠
- 拌入帕玛森芝士粉、盐和黑胡椒，搅匀。酱汁应如浓酸奶般质地，太浓可加几滴水调稀

### 3. 装盘

- 生菜放入大沙拉碗，倒入 2/3 的凯撒酱，翻拌均匀
- 摆上煎鸡胸肉片
- 撒上面包丁和帕玛森芝士薄片
- 淋上剩余酱汁，磨上黑胡椒即可

## 技巧与窍门

- 生菜一定要充分沥干，带水的菜叶酱汁挂不住而且会稀释味道
- 自制凯撒酱的关键是乳化：油要极细极慢地加入，边加边高速搅打
- 如果乳化失败（油水分离），重新取一个蛋黄，把失败的酱汁当油慢慢加入
- 面包丁必须放凉才会变脆，热的时候是软的
- 鸡胸肉煎好后静置可以让肉汁回流，切的时候更多汁
- 介意生蛋黄可以用 65°C 低温水浴蛋（水中煮 13 分钟）代替

## 变体

- **虾仁凯撒沙拉**：鸡肉换成大虾，每面煎 2 分钟即可
- **简易版**：不自制酱汁，用现成的凯撒沙拉酱（超市有售）
- **素食凯撒沙拉**：去掉鸡肉和鳀鱼，加鹰嘴豆和烤南瓜
- **凯撒沙拉卷**：用大号面饼包裹所有食材做成卷饼
