# 腌笃鲜

> "腌"是咸肉，"鲜"是鲜肉，"笃"是小火慢炖，一锅融合咸鲜两味的江南经典｜《很想很想你》第5集

## 基本信息

| 项目 | 内容 |
|------|------|
| 分类 | 很想很想你 |
| 口味 | 咸鲜 |
| 难度 | 中等 |
| 用时 | 约 120 分钟 |
| 份量 | 3-4 人份 |

## 食材

| 食材 | 用量 | 备注 |
|------|------|------|
| 排骨 | 200g | 切小段 |
| 咸肉 | 50g | 提前清水浸泡 2 小时去多余盐分 |
| 火腿 | 50g | 金华火腿最佳 |
| 冬笋 | 150g | 切滚刀块，需焯水去涩 |
| 百叶结 | 100g | |
| 小葱 | 3 根 | 打结 |
| 姜 | 20g | 拍碎 |

## 调料

| 调料 | 用量 | 备注 |
|------|------|------|
| 料酒 | 2 大勺 | 去腥 |
| 盐 | 酌量 | 咸肉火腿已有咸味，先尝后放 |

## 用具

- 砂锅或炖锅
- 汤锅（焯水用）
- 菜刀 + 砧板

## 做法

### 1. 准备工作

- 咸肉提前用清水浸泡 2-3 小时，洗净后切块
- 火腿切小块
- 排骨冷水下锅焯水，大火烧开后煮 3 分钟，捞出冲洗干净
- 冬笋切滚刀块，焯水 2 分钟去草酸和涩味，捞出
- 小葱打结，姜拍碎

### 2. 烹饪步骤

- 砂锅加入足量清水（约 2 升），放入排骨、咸肉、火腿
- 加入葱结、姜块和料酒
- 大火烧开后撇去浮沫，转小火慢炖 1 小时
- 加入冬笋和百叶结，继续小火炖 30 分钟
- 捞出葱结和姜块，尝汤后酌情加盐（通常不需要额外加盐）

### 3. 装盘

- 连锅上桌，或盛入大汤碗中

## 技巧与窍门

- 咸肉一定要提前浸泡去盐，否则汤会过咸
- 全程小火慢炖（"笃"就是小火焖的意思），汤色才清亮
- 冬笋后放是为了保持脆嫩口感，提前放容易炖烂
- 一般不需要额外加盐，咸肉和火腿的咸鲜味已经够了
- 汤和肉都要吃，营养主要在肉里，不要只喝汤弃肉

## 变体

- **春笋版**：春天用春笋代替冬笋，鲜味更突出
- **加豆腐版**：最后 10 分钟加入嫩豆腐块，吸满汤汁格外鲜美
