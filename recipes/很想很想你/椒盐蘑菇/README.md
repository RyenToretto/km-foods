# 椒盐蘑菇

> 外酥里嫩的炸蘑菇裹上椒盐，香气扑鼻，追剧必备的小食｜《很想很想你》第20集

## 基本信息

| 项目 | 内容 |

| 食材 | 用量 | 备注 |

| 平菇或杏鲍菇 | 300g | 撕成条或切厚片 |

| 面粉 | 3 大勺 | |

| 蒜 | 4 瓣 | 切末 |
| 小米辣 | 2 根 | 切圈，可选 |

## 调料

| 调料 | 用量 | 备注 |
|------|------|------|
| 椒盐粉 | 1 大勺 | 或花椒粉+盐自配 |
| 盐 | 1/4 小勺 | 面糊中加少许 |
| 白胡椒粉 | 1/4 小勺 | |
| 食用油 | 适量 | 炸蘑菇用 |

## 用具

- 炒锅或炸锅
- 漏勺
- 搅拌碗
- 厨房纸巾

## 做法

### 1. 准备工作

- 蘑菇洗净沥干水分，平菇撕成长条，杏鲍菇切 1cm 厚片
- 鸡蛋打入碗中，加面粉、淀粉、盐、白胡椒粉和少许水，搅成浓稠面糊（挂在筷子上缓缓滴落的程度）
- 蒜切末，小米辣切圈

### 2. 烹饪步骤

- 蘑菇放入面糊中裹匀，确保每一块都沾满面糊
- 锅中倒油，烧至六成热（筷子入油有小气泡，约 160°C）
- 分批放入蘑菇，中火炸至表面金黄酥脆（约 3-4 分钟），捞出沥油
- 油温升至七成热后，将蘑菇回锅复炸 30 秒，逼出多余油分使外皮更脆，捞出放厨房纸上吸油
- 锅中留 1 大勺底油，中火爆香蒜末和小米辣
- 倒入炸好的蘑菇，撒入椒盐粉，大火快速翻炒均匀（约 30 秒）

### 3. 装盘

- 出锅装盘，撒上葱花

## 技巧与窍门

- 蘑菇一定要沥干水分再裹面糊，否则面糊挂不住容易脱落
- 面糊不要太稀，浓稠些炸出来才有酥脆外壳
- 复炸是酥脆的秘诀：第一次中火炸熟，第二次高温快炸逼油定型
- 椒盐最后撒，趁热翻拌才能均匀附着

## 变体

- **空气炸锅版**：裹好面糊的蘑菇喷少许油，180°C 炸 12 分钟，翻面一次，更健康
- **孜然版**：椒盐换成孜然粉+辣椒粉，烧烤风味
