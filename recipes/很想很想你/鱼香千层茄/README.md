# 鱼香千层茄

> 外酥内软的煎茄夹裹上鱼香酱汁，花生碎点睛，一口层次分明｜《很想很想你》第1集

## 基本信息

| 项目 | 内容 |
|------|------|
| 分类 | 很想很想你 |
| 口味 | 酸甜微辣 |
| 难度 | 中等 |
| 用时 | 约 30 分钟 |
| 份量 | 2 人份 |

## 食材

| 食材 | 用量 | 备注 |
|------|------|------|
| 长茄子 | 1 根（约 250g） | 选细长的紫皮茄子 |
| 鸡蛋 | 1 个 | 做面糊用 |
| 面粉 | 3 大勺 | |
| 蒜 | 8 瓣 | 切末 |
| 小米辣 | 4 根 | 切圈 |
| 花生碎 | 1 大勺 | 装饰提香 |
| 葱花 | 1 大勺 | 装饰 |

## 调料

| 调料 | 用量 | 备注 |
|------|------|------|
| 生抽 | 2 大勺 | |
| 醋 | 1 大勺 | 香醋或米醋均可 |
| 白糖 | 1/2 大勺 | |
| 胡椒粉 | 1/4 小勺 | |
| 鸡精 | 1/2 小勺 | |
| 清水 | 3 大勺 | 混入酱汁中 |
| 食用油 | 适量 | 煎茄子用，约 4 大勺 |

## 用具

- 平底锅或不粘锅
- 菜刀 + 砧板
- 搅拌碗

## 做法

### 1. 准备工作

- 茄子切夹刀片：用两根筷子夹住茄子两侧，45 度斜切一刀不切断，再垂直切一刀切断，形成相连的夹片
- 将生抽、醋、白糖、胡椒粉、鸡精、清水混合成鱼香碗汁备用
- 鸡蛋打入面粉中，加少许水搅拌成比牛奶略稠的面糊
- 蒜切末，小米辣切圈

### 2. 烹饪步骤

- 茄夹先薄蘸一层干面粉，再均匀裹上面糊
- 平底锅倒入食用油，中小火烧至五成热（筷子入油有小气泡）
- 放入茄夹，小火煎至两面金黄酥脆（每面约 2-3 分钟），盛出
- 锅中留少许底油，中火爆香蒜末和小米辣至蒜香四溢（约 30 秒）
- 倒入煎好的茄夹翻炒均匀
- 沿锅边淋入碗汁，大火翻炒收汁至酱汁浓稠裹在茄夹上

### 3. 装盘

- 出锅装盘，撒上花生碎和葱花

## 技巧与窍门

- 夹刀切法是关键：筷子固定防切断，一斜一直形成相连片状
- 面糊不要太稠，薄薄一层即可，太厚吃起来面粉味重
- 煎茄子时油温不能太高，小火慢煎才能外酥内软
- 碗汁可以先尝一下咸淡，不够咸再加少许盐

## 变体

- **免煎版**：茄子不裹面糊，直接切片蒸熟后浇鱼香汁，更清淡
- **肉馅版**：在茄夹中间夹入猪肉馅，煎后再烧，更丰富
