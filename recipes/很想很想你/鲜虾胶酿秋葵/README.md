# 鲜虾胶酿秋葵

> 鲜虾弹牙搭配秋葵清香，蒸出来的健康美味，剧中男女主感情升温之菜｜《很想很想你》第1集

## 基本信息

| 项目 | 内容 |
|------|------|
| 分类 | 很想很想你 |
| 口味 | 咸鲜 |
| 难度 | 中等 |
| 用时 | 约 40 分钟（含冷藏时间 90 分钟） |
| 份量 | 2-3 人份 |

## 食材

| 食材 | 用量 | 备注 |
|------|------|------|
| 鲜虾 | 200g | 去壳去虾线 |
| 秋葵 | 100g（约 10 根） | 选粗细均匀的 |
| 鸡蛋清 | 1 个 | 增加弹性 |
| 葱 | 2 根 | 取葱白泡水用 |
| 姜 | 3 片 | 泡水用 |

## 调料

| 调料 | 用量 | 备注 |
|------|------|------|
| 料酒 | 1 大勺 | 去腥 |
| 盐 | 1/2 小勺 | |
| 白胡椒粉 | 1/4 小勺 | |
| 香油 | 1 小勺 | 提香，最后加 |
| 泰式甜辣酱 | 适量 | 蘸食用，可选 |

## 用具

- 蒸锅
- 菜刀 + 砧板
- 搅拌碗

## 做法

### 1. 准备工作

- 葱白和姜片放入小碗中，加 3 大勺温水浸泡 10 分钟，制成葱姜水
- 鲜虾去壳去虾线，用刀背拍扁后反复剁成细腻的虾蓉
- 虾蓉中分次加入葱姜水（约 2 大勺），同方向搅拌上劲
- 加入料酒、盐、白胡椒粉搅匀，再加蛋清继续搅拌至黏稠有弹性
- 最后加入香油拌匀，盖保鲜膜放冰箱冷藏 90 分钟定型
- 秋葵洗净，纵向对半切开

### 2. 烹饪步骤

- 取出冷藏好的虾胶，用小勺填入秋葵凹槽中，表面抹平
- 将酿好的秋葵整齐摆入蒸盘
- 蒸锅水烧开后放入蒸盘，中火蒸 20 分钟

### 3. 装盘

- 蒸好后取出，整齐码放在盘中
- 旁边放一小碟泰式甜辣酱蘸食

## 技巧与窍门

- 虾蓉一定要同方向搅拌上劲，口感才 Q 弹
- 葱姜水分次加入，边加边搅，虾胶才不会出水
- 冷藏步骤不能省，虾胶定型后更容易填入秋葵且蒸后不散
- 秋葵表面可以薄薄扑一层干淀粉再填虾胶，粘合更牢固

## 变体

- **蒜蓉蒸版**：蒸好后淋上蒜蓉酱油汁（蒜蓉+蒸鱼豉油+少许糖+热油泼香）
- **煎版**：虾胶面朝下放入平底锅，中小火煎至金黄，再翻面煎 2 分钟
