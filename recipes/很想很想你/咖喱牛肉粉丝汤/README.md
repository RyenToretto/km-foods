# 咖喱牛肉粉丝汤

> 浓郁咖喱汤底配上软烂牛肉和爽滑粉丝，暖身暖胃的一碗｜《很想很想你》第2集

## 基本信息

| 项目 | 内容 |
|------|------|
| 分类 | 很想很想你 |
| 口味 | 咖喱咸香 |
| 难度 | 中等 |
| 用时 | 约 120 分钟（含炖牛肉） |
| 份量 | 2-3 人份 |

## 食材

| 食材 | 用量 | 备注 |
|------|------|------|
| 牛腩 | 400g | 切 3cm 见方块 |
| 龙口粉丝 | 1 把（约 80g） | 冷水泡软 |
| 生姜 | 20g | 切片 |
| 大葱 | 1 段 | 切段 |
| 香叶 | 2 片 | |
| 八角 | 1 个 | |
| 香菜 | 2 根 | 切碎，装盘用 |

## 调料

| 调料 | 用量 | 备注 |
|------|------|------|
| 咖喱粉 | 2 大勺 | 或咖喱块 1 块 |
| 盐 | 1 小勺 | |
| 料酒 | 2 大勺 | 焯水去腥 |
| 沙茶酱 | 1/4 小勺 | 可选，增加回味的微辣感 |

## 用具

- 炖锅
- 汤锅
- 漏勺

## 做法

### 1. 准备工作

- 牛腩切块，冷水下锅，加料酒和姜片，大火烧开后煮 3 分钟，捞出冲洗干净血沫
- 龙口粉丝用冷水浸泡 15 分钟至变软，沥干
- 香菜切碎备用

### 2. 烹饪步骤

- 炖锅加入足量清水（约 1.5 升），放入焯好的牛腩、姜片、葱段、香叶、八角
- 大火烧开后转小火慢炖 90 分钟，至牛肉用筷子能轻松扎透
- 捞出牛肉块和适量牛肉汤（约 800ml）到另一锅中
- 大火烧开后加入咖喱粉，搅拌均匀至汤色金黄，煮 5 分钟让咖喱充分融合
- 加入泡好的粉丝，煮 2 分钟至粉丝透明
- 加盐调味，可加沙茶酱增加层次

### 3. 装盘

- 盛入大碗，撒上香菜碎，趁热食用

## 技巧与窍门

- 牛肉一定要冷水下锅焯水，热水下锅会使表面蛋白质迅速凝固，血水锁在里面
- 炖牛肉全程小火，大火炖出来的汤会浑浊且肉易柴
- 咖喱粉要在汤沸腾时加入，搅拌均匀才不会结块
- 粉丝不要煮太久，2 分钟足够，久了会烂成糊
- 剩余的牛肉汤可以留着做其他汤底，冷藏可保存 3 天

## 变体

- **咖喱牛肉面版**：粉丝换成细面条或乌冬面
- **浓咖喱版**：用咖喱块代替咖喱粉，加椰浆 100ml，口感更浓郁
