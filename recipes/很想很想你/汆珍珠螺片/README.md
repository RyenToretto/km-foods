# 汆珍珠螺片

> 螺片汆水如珍珠般卷曲，鲜嫩脆弹，搭配时蔬清爽可口｜《很想很想你》第4集

## 基本信息

| 项目 | 内容 |
|------|------|
| 分类 | 很想很想你 |
| 口味 | 咸鲜 |
| 难度 | 中等 |
| 用时 | 约 20 分钟 |
| 份量 | 2 人份 |

## 食材

| 食材 | 用量 | 备注 |
|------|------|------|
| 鲜海螺肉 | 200g | 切薄片，越薄越好 |
| 青椒 | 1 个 | 切菱形片 |
| 红椒 | 半个 | 切菱形片，配色用 |
| 姜 | 3 片 | 切丝 |
| 蒜 | 3 瓣 | 切片 |
| 葱 | 2 根 | 切段 |

| 调料 | 用量 | 备注 |
|------|------|------|
| 料酒 | 2 大勺 | 汆水去腥 |
| 盐 | 1/2 小勺 | |
| 白糖 | 1/4 小勺 | 提鲜 |
| 蚝油 | 1 大勺 | |

| 水淀粉 | 1 大勺 | 勾薄芡用 |

## 用具

- 炒锅
- 汤锅（汆水用）
- 漏勺

## 做法

### 1. 准备工作

- 海螺肉洗净，斜刀切成约 2mm 的薄片（越薄汆水后卷曲效果越好）
- 青椒、红椒切菱形片，姜切丝，蒜切片，葱切段
- 调好水淀粉备用

### 2. 烹饪步骤

- 汤锅烧开水，加入料酒和姜片
- 放入螺片快速汆烫，见螺片边缘卷起如珍珠状立即捞出（约 10-15 秒，切忌久煮）
- 炒锅倒入食用油，大火烧热
- 爆香姜丝和蒜片（约 20 秒）
- 下青椒和红椒片大火翻炒至断生（约 1 分钟）
- 倒入汆好的螺片，快速翻炒
- 加盐、白糖、蚝油、生抽调味
- 淋入水淀粉勾薄芡，翻匀收汁

### 3. 装盘

- 出锅装盘，撒上葱段点缀

## 技巧与窍门

- 螺片切得越薄，汆水后卷曲越像珍珠，这是这道菜的精髓
- 汆水时间严格控制在 15 秒以内，过久螺肉会变硬变老
- 全程大火快炒，从下螺片到出锅不超过 1 分钟
- 水淀粉勾薄芡即可，太浓影响口感

## 变体

- **上汤版**：不炒，汆好的螺片直接放入清鸡汤中，加枸杞和姜丝，做成清汤
- **XO 酱炒版**：用 XO 酱代替蚝油生抽，风味更浓郁
