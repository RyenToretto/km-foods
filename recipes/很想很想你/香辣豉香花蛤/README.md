# 香辣豉香花蛤

> 花蛤配豆豉辣椒爆炒，鲜辣过瘾，下酒下饭的海鲜快手菜｜《很想很想你》第2集

## 基本信息

| 项目 | 内容 |
|------|------|
| 分类 | 很想很想你 |
| 口味 | 香辣 |
| 难度 | 简单 |
| 用时 | 约 15 分钟（不含吐沙时间） |
| 份量 | 2 人份 |

## 食材

| 食材 | 用量 | 备注 |
|------|------|------|
| 花蛤 | 500g | 提前盐水浸泡 2 小时吐沙 |
| 蒜 | 6 瓣 | 切末 |
| 生姜 | 3 片 | 切丝 |
| 小米辣 | 3 根 | 斜切段 |
| 葱 | 2 根 | 切段，葱白葱绿分开 |

## 调料

| 调料 | 用量 | 备注 |
|------|------|------|
| 豆豉 | 1 大勺 | 用刀稍微剁碎 |
| 料酒 | 2 大勺 | 去腥 |
| 蚝油 | 1 大勺 | |
| 生抽 | 1 大勺 | |
| 白糖 | 1/2 小勺 | 提鲜 |
| 食用油 | 2 大勺 | |

## 用具

- 炒锅
- 锅铲
- 漏勺

## 做法

### 1. 准备工作

- 花蛤放入加盐和几滴香油的清水中浸泡 2 小时，使其充分吐沙
- 烧一锅水，水开后放入花蛤焯至刚刚开口，立即捞出沥干（不要等全部开口，否则肉老）
- 蒜切末，姜切丝，小米辣切段，葱白葱绿分开切段
- 豆豉稍微剁碎

### 2. 烹饪步骤

- 锅中倒入食用油，中火烧热
- 下葱白、姜丝、蒜末、小米辣爆香至蒜末微黄（约 30 秒）
- 加入豆豉翻炒出豉香（约 20 秒）
- 倒入花蛤，大火翻炒均匀
- 沿锅边烹入料酒，加蚝油、生抽、白糖快速翻炒
- 大火收汁至汤汁浓稠裹在花蛤上（约 1 分钟）

### 3. 装盘

- 出锅装盘，撒上葱绿段即可

## 技巧与窍门

- 花蛤吐沙是关键，水中加盐和香油模拟海水环境，吐沙更彻底
- 焯水时花蛤刚开口就捞，过度加热肉质会缩水变硬
- 豆豉要先炒出香味再下花蛤，否则豉香出不来
- 全程大火快炒，花蛤已经焯过水，炒的目的是裹汁入味

## 变体

- **蒜蓉粉丝版**：花蛤摆盘铺粉丝，蒜蓉酱汁浇上后蒸 8 分钟
- **啤酒版**：炒到一半加入 200ml 啤酒焖煮 2 分钟，风味独特
