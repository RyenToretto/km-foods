# 糟辣脆皮鱼

> 贵州传统名菜，外皮金黄酥脆，糟辣酱汁酸辣开胃，壳脆肉嫩｜《很想很想你》第8集

## 基本信息

| 项目 | 内容 |
|------|------|
| 分类 | 很想很想你 |
| 口味 | 酸辣微甜 |
| 难度 | 困难 |
| 用时 | 约 60 分钟（含腌制） |
| 份量 | 3-4 人份 |

## 食材

| 食材 | 用量 | 备注 |
|------|------|------|

| 红薯淀粉 | 3 大勺 | 挂糊用 |
| 姜 | 15g | 切末 |
| 蒜 | 6 瓣 | 切末 |

| 调料 | 用量 | 备注 |

| 糟辣椒 | 3 大勺 | 贵州糟辣椒，核心调味 |
| 盐 | 1 小勺 | 腌鱼用 |
| 料酒 | 2 大勺 | 腌鱼用 |
| 生抽 | 1 大勺 | |
| 白糖 | 1 大勺 | |

## 用具

- 炒锅（深底，炸鱼用）
- 漏勺
- 菜刀 + 砧板

## 做法

### 1. 准备工作

- 鱼处理干净，两面各划 3-4 刀花刀（深至鱼骨但不切断）
- 鱼身内外均匀抹上盐和料酒，放入葱段和姜片，腌制 30 分钟
- 鸡蛋打散，加入红薯淀粉和少许盐搅成浓稠糊状
- 将蛋粉糊均匀涂抹在腌好的鱼全身（包括花刀缝隙中）

### 2. 烹饪步骤

- 锅中倒入食用油，大火烧至七成热（筷子入油有密集气泡，约 180°C）
- 提着鱼尾将鱼滑入油锅，中火炸至鱼身金黄酥脆（约 5-6 分钟），期间用勺子不断舀油浇在鱼身上
- 炸好后捞出沥油，放入盘中
- 锅中留 2 大勺底油，中火爆香姜末和蒜末
- 加入糟辣椒翻炒出红油和酱香（约 1 分钟）
- 加入鸡汤、生抽、白糖、醋，烧开
- 将炸好的鱼放回锅中，中小火焖 5 分钟让鱼入味
- 小心取出鱼装盘，锅中汤汁用水淀粉勾芡至浓稠

### 3. 装盘

- 将勾好芡的糟辣酱汁均匀浇在鱼身上，撒葱花

## 技巧与窍门

- 花刀要深到鱼骨，这样既容易炸透又方便入味
- 蛋粉糊要涂匀涂厚，是"脆皮"的关键
- 炸鱼时油温要够高，下锅后先不要翻动，等底部定型后再轻轻翻面
- 糟辣椒是灵魂调料，如买不到可用剁椒+少许醋代替，但风味会有差别
- 焖的时间不宜过长，5 分钟足够入味且保持脆皮口感

## 变体

- **糟辣脆皮带鱼版**：用带鱼段代替整鱼，更容易操作
- **免炸版**：鱼段煎至两面金黄后直接加糟辣酱汁烧，减少用油
