# 糖醋排骨

> 甜甜蜜蜜，经典糖醋口味——外焦里嫩裹满琥珀色糖醋汁，酸甜开胃大人小孩都爱

## 基本信息

| 项目 | 内容 |
|------|------|
| 分类 | 年夜饭 |
| 口味 | 酸甜微咸 |
| 难度 | 中等 |
| 用时 | 约 45 分钟 |
| 份量 | 4-6 人份 |

## 食材

| 食材 | 用量 | 备注 |
|------|------|------|
| 猪小排 | 500g | 让肉档剁成 3-4cm 小段 |
| 葱 | 2 根 | 切段 |
| 姜 | 3 片 | — |
| 白芝麻 | 1 小勺（约 3g） | 装盘撒面用 |

## 调料

| 调料 | 用量 | 备注 |
|------|------|------|
| 料酒 | 2 大勺（约 30ml） | 焯水去腥 |
| 生抽 | 2 大勺（约 30ml） | — |
| 香醋 | 3 大勺（约 45ml） | 镇江香醋最佳 |
| 白砂糖 | 3 大勺（约 45g） | — |
| 番茄酱 | 1 大勺（约 15g） | 增色增酸，可选 |
| 盐 | 半小勺（约 2g） | — |
| 食用油 | 2 大勺（约 30ml） | 煎排骨用 |
| 清水 | 200ml | 焖煮用 |

## 用具

- 炒锅
- 汤锅（焯水用）
- 漏勺

## 做法

### 1. 准备工作

- 排骨冷水浸泡 30 分钟，泡出血水，中途换水 2 次
- 调糖醋汁：生抽 + 香醋 + 白砂糖 + 番茄酱 + 盐 + 清水 200ml，搅拌均匀备用
- 锅中冷水放入排骨、葱段、姜片、料酒，大火煮沸后撇去浮沫，焯 3 分钟捞出沥干

### 2. 烹饪步骤

- 炒锅中倒入食用油，中火加热，放入焯好的排骨
- 中火煎 5-6 分钟，不断翻面，至排骨表面微微焦黄
- 倒入调好的糖醋汁，大火煮沸
- 转中小火盖盖焖煮 20-25 分钟，中途翻动 2-3 次防止粘底
- 开盖转大火收汁，持续翻炒 3-5 分钟，至汤汁变浓稠、裹满每块排骨
- 汁水收至浓稠发亮、挂在排骨上时关火

### 3. 装盘

- 盛入盘中，撒上白芝麻
- 可用生菜叶垫底增添色彩

## 技巧与窍门

- 排骨冷水浸泡去血水是关键，血水泡干净排骨才不腥
- 焯水要冷水下锅，热水下锅血沫会封在排骨里面
- 煎排骨这一步让表面焦化锁住肉汁，炖好后外焦里嫩
- 糖醋比例经典口诀：3 勺糖 3 勺醋 2 勺酱油，可根据口味微调
- 收汁时要不停翻炒，防止糖汁焦底变苦
- 最后收汁的火候最关键：汁水裹住排骨、色泽琥珀发亮就要立刻关火
- 用镇江香醋比白醋风味更醇厚

## 变体

- **可乐排骨**：糖醋汁换成 1 罐可乐 + 2 大勺生抽，简单版糖醋风味
- **糖醋里脊**：排骨换成里脊肉切条，裹淀粉炸酥后浇糖醋汁
- **菠萝糖醋排骨**：加入菠萝块一起炖，果酸让排骨更嫩更有热带风味
