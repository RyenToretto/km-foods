# 红烧猪蹄

> 发财就手，浓香软糯——寓意新年"横财就手"，猪蹄软烂脱骨、酱汁浓郁，满满的胶原蛋白

## 基本信息

| 项目 | 内容 |
|------|------|
| 分类 | 年夜饭 |
| 口味 | 咸甜浓香 |
| 难度 | 中等 |
| 用时 | 约 90 分钟 |
| 份量 | 4-6 人份 |

## 食材

| 食材 | 用量 | 备注 |
|------|------|------|
| 猪前蹄 | 1 只（约 800g） | 让肉档剁成 4-6 块，前蹄比后蹄肉多 |
| 葱 | 3 根 | 切段 |
| 姜 | 1 块（约 30g） | 切厚片 |
| 大蒜 | 4 瓣 | 拍扁 |
| 干辣椒 | 3 个 | 可选 |

## 调料

| 调料 | 用量 | 备注 |
|------|------|------|
| 生抽 | 3 大勺（约 45ml） | — |
| 老抽 | 2 大勺（约 30ml） | 上色关键 |
| 料酒 | 3 大勺（约 45ml） | 去腥 |
| 冰糖 | 30g | 炒糖色用，比白砂糖上色更亮 |
| 八角 | 2 个 | — |
| 桂皮 | 1 小段（约 3cm） | — |
| 香叶 | 3 片 | — |
| 花椒 | 1 小勺（约 3g） | — |
| 盐 | 1 小勺（约 3g） | 后期调味 |
| 食用油 | 2 大勺（约 30ml） | 炒糖色用 |
| 热水 | 足量 | 没过猪蹄 |

## 用具

- 炒锅（炒糖色用）
- 砂锅或炖锅（炖煮用）
- 汤锅（焯水用）

## 做法

### 1. 准备工作

- 猪蹄洗净，冷水下锅，加入葱段（1 根）、姜片（3 片）、料酒（2 大勺），大火烧开
- 撇去大量浮沫，焯煮 5 分钟后捞出，用温水冲洗干净表面浮沫
- 准备好所有香料

### 2. 烹饪步骤

**炒糖色：**
- 炒锅倒入食用油，放入冰糖，小火慢炒
- 冰糖会先融化变成液体，然后开始冒大泡，最后变成密集小泡、颜色变成琥珀色
- 看到琥珀色立刻放入猪蹄快速翻炒裹色（整个炒糖过程约 3-4 分钟）

**红烧炖煮：**
- 猪蹄裹上糖色后，加入葱段、姜片、蒜瓣、干辣椒、八角、桂皮、香叶、花椒翻炒出香
- 烹入料酒（1 大勺），加入生抽和老抽，翻炒均匀
- 转入砂锅，加入足量热水没过猪蹄
- 大火煮沸后转最小火，盖盖炖煮 60-70 分钟
- 炖至筷子能轻松插入猪蹄皮肉、骨肉可以轻松分离的程度
- 加盐调味，开盖转大火收汁 8-10 分钟，至汤汁浓稠裹住猪蹄

### 3. 装盘

- 猪蹄盛入深盘，浇上浓汁
- 撒少许葱花点缀
- 汤汁放凉后会凝成肉皮冻，拌面拌饭都极佳

## 技巧与窍门

- 炒糖色是红烧猪蹄色泽红亮的关键，一定要小火慢炒，大火会炒焦变苦
- 糖色炒到琥珀色（焦糖色）就要立刻下猪蹄，晚一秒就会炒过发黑发苦
- 炖猪蹄一定要用热水，冷水会让胶原蛋白收缩，猪蹄不容易炖软
- 全程小火慢炖，大火炖汤浑浊且肉质发柴
- 前蹄比后蹄筋和肉更多，口感更好
- 猪蹄焯水要彻底，浮沫撇干净，否则成品有腥味
- 如果时间充裕，炖 90 分钟效果更佳，用高压锅可缩短至 30-40 分钟

## 变体

- **黄豆焖猪蹄**：加入提前泡好的黄豆 100g 一起炖，豆香猪蹄两相宜
- **卤猪蹄**：用卤水（加入更多香料）卤制，出锅后冷吃更弹
- **啤酒猪蹄**：热水换成 1 罐啤酒 + 适量热水，啤酒有嫩肉去腥的效果
