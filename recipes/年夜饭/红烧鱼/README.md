# 红烧鱼

> 年年有余，红烧整鱼——浓油赤酱色泽红亮，鱼肉鲜嫩入味，年夜饭餐桌上的必备好彩头

## 基本信息

| 项目 | 内容 |
|------|------|
| 分类 | 年夜饭 |
| 口味 | 咸鲜微甜 |
| 难度 | 中等 |
| 用时 | 约 40 分钟 |
| 份量 | 4-6 人份 |

## 食材

| 食材 | 用量 | 备注 |
|------|------|------|
| 鲤鱼/鲈鱼 | 1 条（约 750g） | 整条，让鱼档处理好内脏和鳞片 |
| 葱 | 3 根 | 2 根切段，1 根切丝（装盘用） |
| 姜 | 1 块（约 30g） | 切片 |
| 蒜 | 4 瓣 | 拍扁 |
| 干辣椒 | 2-3 个 | 可选，增香不增辣 |

## 调料

| 调料 | 用量 | 备注 |
|------|------|------|
| 生抽 | 3 大勺（约 45ml） | — |
| 老抽 | 1 大勺（约 15ml） | 上色用 |
| 料酒 | 2 大勺（约 30ml） | 去腥 |
| 白砂糖 | 1 大勺（约 15g） | 提鲜增色 |
| 醋 | 1 小勺（约 5ml） | 去腥提鲜 |
| 盐 | 1 小勺（约 3g） | 腌鱼用 |
| 食用油 | 足量（约 50ml） | 煎鱼用 |
| 热水 | 300ml | 烧鱼用 |

## 用具

- 大平底锅或炒锅（能放下整鱼）
- 锅铲
- 厨房纸巾

## 做法

### 1. 准备工作

- 鱼清洗干净，两面各划 3-4 刀斜刀花（深至骨头，间隔 2cm），方便入味
- 用盐和料酒（1 大勺）抹匀鱼身内外，腌制 15 分钟
- 腌好后用厨房纸巾彻底擦干鱼身表面水分（这一步防止煎鱼溅油）
- 葱姜蒜切好备用

### 2. 烹饪步骤

- 锅中倒入食用油，中大火加热至油面微微冒烟（约 180°C）
- 手提鱼尾将鱼轻轻滑入锅中，中火煎 3-4 分钟至底面金黄定型
- 小心翻面，继续煎 3-4 分钟至两面金黄
- 鱼推到锅边，利用锅中底油爆香葱段、姜片、蒜瓣、干辣椒，约 30 秒
- 沿锅边淋入料酒（1 大勺），烹出酒香
- 加入生抽、老抽、白砂糖、醋，轻轻翻动调料使其化开
- 倒入 300ml 热水（一定要用热水，冷水会让鱼肉发紧），水量没过鱼身 2/3
- 大火煮沸后转中小火，盖盖焖烧 12-15 分钟
- 中途用勺舀汤汁浇淋鱼身 3-4 次，使上色均匀
- 开盖转大火收汁 2-3 分钟，至汤汁浓稠挂在鱼身上

### 3. 装盘

- 用锅铲小心将整鱼托起，盛入鱼盘中
- 浇上锅中剩余浓汁
- 表面撒上葱丝，可点缀几粒枸杞增添喜庆感

## 技巧与窍门

- 煎鱼前鱼身一定要擦干，这是不粘锅不破皮的关键
- 油温要够热再下鱼，可以先放一小块姜试油温，姜片周围起泡就说明温度够了
- 煎鱼时不要急着翻面，等底面完全定型金黄后再翻，否则容易破皮
- 烧鱼必须用热水，冷水会导致鱼肉收缩变紧发柴
- 年夜饭的鱼讲究"有头有尾"，一定要整条上桌
- 鱼头朝向家中长辈或贵客，这是传统礼仪

## 变体

- **糖醋鱼**：去掉老抽，增加醋和糖的用量（3 大勺醋 + 3 大勺糖），勾芡后浇汁
- **清蒸鱼**：如果鱼足够新鲜，清蒸更能体现鱼的本味
- **豆瓣鱼**：加 1.5 大勺郫县豆瓣酱爆香，川味十足
