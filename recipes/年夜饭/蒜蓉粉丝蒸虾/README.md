# 蒜蓉粉丝蒸虾

> 笑哈哈（虾），蒜香蒸虾——蒜蓉金黄喷香，虾肉鲜嫩弹牙，粉丝吸满汤汁，年夜饭讨个好口彩

## 基本信息

| 项目 | 内容 |
|------|------|
| 分类 | 年夜饭 |
| 口味 | 蒜香鲜甜 |
| 难度 | 简单 |
| 用时 | 约 25 分钟 |
| 份量 | 4-6 人份 |

## 食材

| 食材 | 用量 | 备注 |
|------|------|------|
| 大虾 | 12 只（约 400g） | 鲜活基围虾或明虾，越大越好 |
| 粉丝 | 1 把（约 50g） | 龙口绿豆粉丝 |
| 大蒜 | 1 整头（约 40g） | 剥好切末 |
| 小葱 | 2 根 | 切葱花，出锅装饰 |
| 红椒 | 半个 | 切小丁，增色用 |

## 调料

| 调料 | 用量 | 备注 |
|------|------|------|
| 生抽 | 2 大勺（约 30ml） | — |
| 蒸鱼豉油 | 1 大勺（约 15ml） | 比生抽更鲜甜 |
| 料酒 | 1 大勺（约 15ml） | 去腥 |
| 白砂糖 | 半小勺（约 2g） | 提鲜 |
| 食用油 | 3 大勺（约 45ml） | 炒蒜蓉用 |
| 盐 | 半小勺（约 2g） | 调味 |

## 用具

- 蒸锅或蒸箱
- 长盘（能摆开虾的）
- 小锅（炒蒜蓉用）
- 剪刀

## 做法

### 1. 准备工作

- 粉丝用冷水泡软（约 15 分钟），剪成 15cm 长段，沥干铺在蒸盘底部
- 大虾清洗干净，剪去虾须虾脚，用剪刀沿虾背剪开（不要剪断），挑去虾线
- 将虾按压展开呈蝴蝶状，整齐排列在粉丝上
- 蒜末准备好，分成两份（一份炒金蒜、一份生蒜末）

### 2. 烹饪步骤

**炒蒜蓉酱：**
- 小锅中倒入食用油，冷油放入一半蒜末（约 20g）
- 小火慢慢炒 2-3 分钟至蒜末变成浅金黄色、香气浓郁
- 关火，加入另一半生蒜末，利用余温拌匀（生熟蒜蓉混合蒜香更有层次）
- 加入生抽、蒸鱼豉油、料酒、白砂糖、盐，搅拌均匀

**蒸制：**
- 将蒜蓉酱均匀地铺在每只虾的背部开口处
- 蒸锅水烧开后，放入虾盘，大火蒸 8-10 分钟（根据虾的大小调整）
- 虾身完全变红、虾肉变白弹牙即熟，不可蒸过久

### 3. 装盘

- 取出蒸盘，撒上葱花和红椒丁
- 另起一锅烧热 1 大勺油，浇在葱花上激发香气（会发出滋啦声）
- 趁热上桌

## 技巧与窍门

- 蒜蓉一定要用生熟混合的方式：炒过的蒜蓉香而不辣，生蒜蓉辣而鲜，两者混合层次最丰富
- 炒蒜蓉用小火慢炒，大火极易炒焦发苦
- 虾背剪开后按压展开，蒸好造型漂亮，而且蒜蓉能塞入虾背更入味
- 粉丝铺底可以吸收虾的鲜汁，比虾本身还好吃
- 蒸的时间不宜过长，大虾 8-10 分钟足够，过久虾肉缩水变老
- 最后浇热油是点睛之笔，能激发蒜蓉和葱花的香气

## 变体

- **蒜蓉粉丝蒸扇贝**：虾换成开壳扇贝，做法完全一样
- **蒜蓉粉丝蒸鲍鱼**：用小鲍鱼替代，更显档次
- **金银蒜蒸排骨**：排骨切小块替代虾，蒸 20 分钟
