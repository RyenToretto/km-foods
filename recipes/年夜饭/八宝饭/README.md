# 八宝饭

> 团团圆圆，甜糯喜庆——八种珍果铺面寓意八方来财，糯米软糯豆沙甜蜜，年夜饭的甜蜜收尾

## 基本信息

| 项目 | 内容 |
|------|------|
| 分类 | 年夜饭 |
| 口味 | 甜糯香润 |
| 难度 | 中等 |
| 用时 | 约 80 分钟 |
| 份量 | 6-8 人份 |

## 食材

| 食材 | 用量 | 备注 |
|------|------|------|
| 圆粒糯米 | 300g | 提前浸泡 4 小时以上 |
| 红豆沙 | 150g | 买现成的或自制均可 |
| 红枣 | 8 颗 | 去核 |
| 葡萄干 | 30g | — |
| 蜜枣 | 4 颗 | 对半切开 |
| 核桃仁 | 30g | 掰成小块 |
| 莲子 | 20g（约 10 颗） | 干莲子提前泡软煮熟 |
| 枸杞 | 15g | 温水泡软 |
| 瓜子仁 | 15g | 可用松子替代 |
| 蜜冬瓜条 | 30g | 切小丁，可选 |

## 调料

| 调料 | 用量 | 备注 |
|------|------|------|
| 白砂糖 | 2 大勺（约 30g） | 拌糯米用 |
| 猪油/黄油 | 2 大勺（约 30g） | 拌糯米用，猪油更香 |
| 桂花糖 | 1 大勺（约 15g） | 最后淋面，增香 |
| 清水 | 100ml | 糖浆勾芡用 |
| 淀粉 | 1 小勺（约 5g） | 糖浆勾芡用 |

## 用具

- 蒸锅
- 大碗（容量约 800ml，作模具）
- 保鲜膜

## 做法

### 1. 准备工作

- 糯米提前浸泡 4 小时以上（最好隔夜泡），沥干水分
- 干莲子温水泡 2 小时，放入小锅加水煮 20 分钟至软糯，沥干
- 红枣去核，枸杞温水泡软，蜜枣对半切开
- 蒸锅烧开水，糯米铺在蒸布上，大火蒸 25-30 分钟至糯米完全熟透、粒粒分明

### 2. 烹饪步骤

**拌糯米：**
- 蒸好的糯米趁热加入白砂糖和猪油（或黄油），翻拌均匀至油脂完全吸收
- 拌好的糯米油亮微甜，放一旁稍晾

**铺碗造型：**
- 取一个大碗，内壁刷一层薄薄的猪油（方便脱模）
- 将红枣、蜜枣、莲子、核桃仁、葡萄干、枸杞、瓜子仁按照喜欢的图案在碗底和碗壁摆好（碗底朝上就是最终成品的表面）
- 小心地铺一层糯米（约 1.5cm 厚），用勺子背压实贴紧碗壁，让果料不移位
- 中间放入红豆沙铺平
- 再盖上一层糯米封顶，压紧实，表面抹平

**蒸制：**
- 碗口盖上保鲜膜或扣一个盘子
- 放入蒸锅，大火蒸 30-40 分钟至糯米与豆沙充分融合

**制作糖浆：**
- 100ml 清水加 1 大勺白砂糖煮沸
- 淋入水淀粉勾薄芡，加入桂花糖搅匀

### 3. 装盘

- 蒸好的八宝饭取出，倒扣在圆盘上，揭开碗
- 果料图案朝上，淋上桂花糖浆
- 整体色彩缤纷、圆润饱满，寓意团圆美满

## 技巧与窍门

- 糯米一定要提前泡足时间，泡不透蒸出来会有硬芯
- 猪油是传统八宝饭香气的灵魂，用黄油替代也可以但风味不同
- 碗壁刷油是成功脱模的关键，不刷油糯米会粘在碗上
- 摆果料时碗底是最终的"正面"，要摆得好看有规律
- 铺糯米时一定要压实，尤其是碗壁的部分，否则倒扣出来会散
- 八宝饭可以提前做好冷冻保存，吃的时候回蒸 15-20 分钟即可
- "八宝"取吉利数字八，果料种类可以根据喜好调整，但要凑满八种

## 变体

- **紫薯八宝饭**：红豆沙换成紫薯泥，颜色更梦幻
- **南瓜八宝饭**：糯米中拌入南瓜泥，金黄色寓意金玉满堂
- **迷你八宝饭**：用小碗做成一人份的小八宝饭，方便分食
