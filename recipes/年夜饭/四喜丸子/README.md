# 四喜丸子

> 四季平安，红烧大肉丸——寓意福禄寿喜四大喜事，外酥里嫩汤汁浓郁，年夜饭的压轴硬菜

## 基本信息

| 项目 | 内容 |
|------|------|
| 分类 | 年夜饭 |
| 口味 | 咸鲜浓香 |
| 难度 | 中等 |
| 用时 | 约 60 分钟 |
| 份量 | 4-6 人份（4 个大丸子） |

## 食材

| 食材 | 用量 | 备注 |
|------|------|------|
| 猪前腿肉 | 500g | 肥瘦比 3:7，让肉档绞成粗肉馅 |
| 鸡蛋 | 1 个 | 增加黏性 |
| 葱 | 2 根 | 切末 |
| 姜 | 1 小块（约 15g） | 切末 |
| 馒头 | 半个（约 40g） | 掰成小粒泡水挤干，增加松软口感 |
| 荸荠 | 4 个（约 60g） | 去皮切碎丁，增加脆感 |
| 上海青 | 4-6 棵 | 摆盘装饰用 |

## 调料

| 调料 | 用量 | 备注 |
|------|------|------|
| 生抽 | 2 大勺（约 30ml） | — |
| 老抽 | 1 大勺（约 15ml） | 上色用 |
| 蚝油 | 1 大勺（约 15ml） | 提鲜 |
| 料酒 | 1 大勺（约 15ml） | 去腥 |
| 盐 | 1 小勺（约 3g） | — |
| 白胡椒粉 | 半小勺（约 1g） | 去腥增香 |
| 五香粉 | 半小勺（约 1g） | — |
| 淀粉 | 2 大勺（约 20g） | 拌肉馅用 |
| 花生油 | 足量（约 500ml） | 炸丸子用 |
| 高汤或热水 | 400ml | 炖煮用 |
| 水淀粉 | 1 大勺淀粉 + 3 大勺水 | 最后勾芡用 |

## 用具

- 大碗（拌馅用）
- 炒锅或深锅（炸丸子 + 炖煮）
- 砂锅（可选，炖煮更入味）

## 做法

### 1. 准备工作

- 猪肉馅放入大碗，加盐、生抽（1 大勺）、料酒、白胡椒粉、五香粉
- 打入鸡蛋，加葱末、姜末
- 馒头泡水挤干碎成小粒加入，荸荠碎丁加入
- 加入淀粉，用手沿同一方向搅拌上劲，至肉馅抱团有弹性（约搅拌 3-5 分钟）
- 将肉馅摔打几次（抓起肉馅往碗里摔，反复 10-15 次），使肉馅更紧实

### 2. 烹饪步骤

**炸丸子：**
- 肉馅分成 4 等份，双手抹少许油，将每份团成直径约 6-7cm 的大丸子
- 锅中倒入花生油，中火加热至六成热（约 170°C，筷子入油有密集小泡）
- 将丸子轻轻滑入油锅，中火炸 5-6 分钟，期间轻轻翻动
- 炸至表面金黄定型捞出沥油

**红烧炖煮：**
- 锅留底油 1 大勺，放入葱段姜片爆香
- 加入高汤或热水 400ml，加生抽（1 大勺）、老抽、蚝油，煮沸
- 放入炸好的丸子，大火煮开后转小火，盖盖炖煮 30-35 分钟至丸子充分入味
- 中途翻动 1-2 次使上色均匀
- 另起一锅烧开水，加少许盐和油，焯熟上海青捞出备用
- 丸子炖好后捞出摆入盘中，锅中汤汁大火收至浓稠
- 淋入水淀粉勾薄芡，汤汁变透明浓稠后浇在丸子上

### 3. 装盘

- 4 个丸子居中摆放，焯好的上海青围一圈
- 浇上浓稠酱汁，色泽红亮诱人

## 技巧与窍门

- 肥瘦 3:7 的比例最佳，太瘦丸子发柴，太肥则油腻
- 搅拌上劲和摔打是丸子弹嫩不散的关键，不能偷懒
- 馒头粒吸收汤汁后让丸子更加松软多汁
- 荸荠碎丁提供脆爽口感，是四喜丸子区别于普通肉丸的特色
- 炸丸子油温不要太高，中火慢炸才能外酥里熟不焦糊
- 炖煮时间要足够，小火慢炖才能让丸子由外及内充分入味
- 四喜丸子一定是 4 个，寓意"福禄寿喜"四大喜事

## 变体

- **清蒸狮子头**：不炸不红烧，肉馅团好后直接放入砂锅加高汤小火炖 1.5 小时，清淡版
- **蟹粉狮子头**：肉馅中加入蟹肉和蟹黄，淮扬菜经典做法
- **荠菜丸子**：加入切碎的荠菜，春天时令风味
