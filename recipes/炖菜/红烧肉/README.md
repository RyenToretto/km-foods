# 红烧肉

> 肥而不腻色泽红亮的经典硬菜，入口即化，浓油赤酱

## 基本信息

| 项目 | 内容 |
|------|------|
| 分类 | 炖菜 |
| 口味 | 浓香偏甜（本帮风格） |
| 难度 | 中等 |
| 用时 | 约 90 分钟 |
| 份量 | 3-4 人份 |

## 食材

| 食材 | 用量 | 备注 |
|------|------|------|
| 五花肉 | 500g | 选皮薄、层次分明的五花肉 |
| 鸡蛋 | 4 个 | 煮熟去壳，可选 |
| 姜 | 3 片（约 10g） | 拍扁 |
| 大葱 | 1 根 | 切段 |

## 调料

| 调料 | 用量 | 备注 |
|------|------|------|
| 冰糖 | 30g | 炒糖色用，比白糖色泽更亮 |
| 生抽 | 2 汤匙（约 30ml） | — |
| 老抽 | 1 汤匙（约 15ml） | 上色用 |
| 料酒 | 3 汤匙（约 45ml） | — |
| 八角 | 2 个 | — |
| 桂皮 | 1 小段（约 3cm） | — |
| 香叶 | 2 片 | — |
| 食用油 | 1 汤匙（约 15ml） | 炒糖色用 |
| 开水 | 约 500ml | 炖煮用 |

## 用具

- 炒锅或铸铁锅
- 砂锅（慢炖用，可选）

## 做法

### 1. 准备工作

- 五花肉冷水下锅，加料酒 1 汤匙、姜片 2 片，大火煮开后撇去浮沫，煮 3 分钟捞出
- 冲洗干净后切成 3cm 见方的块，用厨房纸吸干表面水分
- 鸡蛋煮熟去壳，表面用牙签扎几个小孔（方便入味）

### 2. 烹饪步骤

- **炒糖色**：锅中加油和冰糖，**小火**慢慢加热，不断搅拌。冰糖先融化冒大泡（水分蒸发），再变小泡转为琥珀色时，立即下入五花肉翻炒上色（整个过程约 3-4 分钟，颜色变深立刻下肉，晚了会发苦）
- 翻炒肉块使每面均匀裹上糖色，约 2 分钟
- 加入葱段、姜片、八角、桂皮、香叶，翻炒出香味
- 倒入料酒沿锅边烹入，翻炒去腥
- 加入生抽、老抽，翻炒均匀
- 加入开水（没过肉面约 1cm），大火烧开后撇去浮沫
- **转小火，盖盖慢炖 50-60 分钟**，中途放入鸡蛋一起炖
- 最后 10 分钟打开锅盖，**转中大火收汁**，不断翻动防止粘锅，汤汁浓稠裹满肉块即可

### 3. 装盘

- 将红烧肉和卤蛋盛入碗中
- 浇上锅中剩余汤汁
- 可点缀少许葱花

## 技巧与窍门

- 炒糖色是红烧肉的灵魂，全程小火、不停搅拌，颜色到琥珀色立刻下肉
- 一定用开水炖，冷水会使肉收缩变紧发柴
- 小火慢炖时间要够，肥肉的胶原蛋白充分溶解才能达到"肥而不腻"
- 收汁时要守在锅边翻动，糖色容易糊底
- 五花肉焯水后吸干水分再炒糖色，否则下锅会炸油

## 变体

- **毛氏红烧肉**：不加酱油，只用糖色上色，加干辣椒和豆豉
- **腐乳红烧肉**：加 2 块红腐乳碾碎拌入，色泽更红，风味独特
- **梅菜扣肉**：红烧肉切薄片，与梅干菜同蒸 2 小时
- **啤酒红烧肉**：开水换成一罐啤酒（330ml），风味更温和
