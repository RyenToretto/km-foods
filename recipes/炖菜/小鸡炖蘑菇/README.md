# 小鸡炖蘑菇

> 东北经典炖菜，鸡肉鲜嫩菌香浓郁，配上宽粉更是一绝

## 基本信息

| 项目 | 内容 |
|------|------|
| 分类 | 炖菜 |
| 口味 | 咸鲜 |
| 难度 | 中等 |
| 用时 | 约 80 分钟 |
| 份量 | 4-5 人份 |

## 食材

| 食材 | 用量 | 备注 |
|------|------|------|
| 三黄鸡/小笨鸡 | 1 只（约 1000g） | 剁成 4cm 大小的块 |
| 干榛蘑 | 50g | 东北野生榛蘑最佳 |
| 宽粉条 | 100g | 红薯粉条或土豆粉条 |
| 姜 | 4 片（约 15g） | 拍扁 |
| 大葱 | 2 根 | 切段 |
| 蒜 | 4 瓣（约 15g） | 拍扁 |
| 干辣椒 | 3 个 | 不吃辣可省略 |

## 调料

| 调料 | 用量 | 备注 |
|------|------|------|
| 生抽 | 3 汤匙（约 45ml） | — |
| 老抽 | 1 汤匙（约 15ml） | 上色 |
| 料酒 | 3 汤匙（约 45ml） | — |
| 冰糖 | 15g | — |
| 八角 | 2 个 | — |
| 桂皮 | 1 小段（约 3cm） | — |
| 花椒 | 1 茶匙（约 2g） | — |
| 盐 | 1 茶匙（约 6g） | 最后调味 |

| 开水 | 约 800ml | — |

## 用具

- 炒锅
- 铸铁锅或砂锅（炖煮用）

## 做法

### 1. 准备工作

- 干榛蘑用温水浸泡 **30 分钟**至完全泡发，洗净泥沙（泡蘑菇的水留用，沉淀后取上层清液）
- 宽粉条用温水泡软（约 15 分钟），剪成 20cm 左右长段
- 鸡肉剁块后冷水下锅，加料酒 1 汤匙、姜片 2 片，大火煮开撇去浮沫，煮 2 分钟捞出洗净

### 2. 烹饪步骤

- 炒锅烧热加油，放入冰糖，**小火炒至琥珀色**
- 下入鸡块快速翻炒上色，约 2-3 分钟
- 加入葱段、姜片、蒜、八角、桂皮、花椒、干辣椒，翻炒出香味
- 烹入料酒，加生抽、老抽，翻炒均匀
- 加入泡发好的榛蘑，翻炒 1 分钟
- 倒入开水和蘑菇浸泡液（取澄清部分），水量没过食材
- 大火烧开后转入砂锅，**小火慢炖 40-50 分钟**
- 鸡肉软烂后加入泡好的粉条，**继续炖 10-15 分钟**至粉条透明入味
- 加盐调味，大火收汁 2-3 分钟

### 3. 装盘

- 直接用砂锅上桌，保温效果好
- 撒上香菜段和葱花

## 技巧与窍门

- 榛蘑是这道菜的灵魂，建议用东北干榛蘑而非新鲜蘑菇，干蘑菇的浓缩鲜味无法替代
- 泡蘑菇的水一定要利用，含有大量鲜味物质，但底部泥沙要丢弃
- 选用小笨鸡（走地鸡）比肉鸡口感和香味好很多
- 粉条最后放，太早放会炖烂碎掉
- 这道菜隔夜后更入味，适合提前做好第二天热着吃

## 变体

- **小鸡炖粉条**：不放蘑菇，增加粉条用量，更朴素的家常版本
- **黄焖鸡**：不放蘑菇和粉条，加青椒土豆，调味偏重酱香
- **松茸炖鸡汤**：把榛蘑换松茸，不炒糖色，做成清炖汤品
- **板栗炖鸡**：加入去壳板栗 200g 同炖，秋季时令做法
