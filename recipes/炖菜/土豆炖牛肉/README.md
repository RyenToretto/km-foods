# 土豆炖牛肉

> 软烂入味的家常炖菜，汤汁浓郁，冬日暖胃首选

## 基本信息

| 项目 | 内容 |
|------|------|
| 分类 | 炖菜 |
| 口味 | 咸香 |
| 难度 | 简单 |
| 用时 | 约 100 分钟 |
| 份量 | 3-4 人份 |

## 食材

| 食材 | 用量 | 备注 |
|------|------|------|
| 牛腩/牛腱 | 500g | 切 3cm 方块，牛腩更适合炖 |
| 土豆 | 2 个（约 400g） | 去皮切滚刀块 |
| 胡萝卜 | 1 根（约 150g） | 切滚刀块 |
| 洋葱 | 半个（约 100g） | 切大块 |
| 姜 | 4 片（约 15g） | 拍扁 |
| 大葱 | 1 根 | 切段 |

## 调料

| 调料 | 用量 | 备注 |
|------|------|------|
| 生抽 | 3 汤匙（约 45ml） | — |
| 老抽 | 1 汤匙（约 15ml） | 上色 |
| 料酒 | 3 汤匙（约 45ml） | — |
| 豆瓣酱 | 1 汤匙（约 15g） | 郫县豆瓣，剁细 |
| 番茄酱 | 1 汤匙（约 15g） | 增色增酸，可选 |
| 冰糖 | 10g | 约 3-4 小粒 |
| 八角 | 2 个 | — |
| 桂皮 | 1 小段（约 3cm） | — |
| 干辣椒 | 2 个 | 不吃辣可省略 |
| 食用油 | 2 汤匙（约 30ml） | — |
| 盐 | 1 茶匙（约 6g） | 出锅前调整 |
| 开水 | 约 800ml | 没过食材 |

## 用具

- 炒锅
- 砂锅或汤锅（炖煮用）

## 做法

### 1. 准备工作

- 牛肉冷水下锅，加料酒 1 汤匙、姜片 2 片，大火煮开后撇去浮沫，继续煮 3 分钟
- 捞出牛肉冲洗干净，沥干水分
- 土豆去皮切滚刀块，泡在清水中防氧化变黑
- 胡萝卜切滚刀块，洋葱切大块

### 2. 烹饪步骤

- 炒锅热油，下入牛肉块，**中大火煎至表面微黄**，约 3 分钟，盛出
- 锅中留底油，**小火炒香豆瓣酱**至出红油（约 1 分钟）
- 加入葱段、姜片、八角、桂皮、干辣椒，炒出香味
- 放入洋葱块翻炒至变软
- 倒入牛肉，加生抽、老抽、料酒、番茄酱、冰糖翻炒均匀
- 加入开水没过食材，大火烧开后撇沫
- **转小火炖 60 分钟**（牛肉用筷子能轻松插入即可）
- 加入土豆和胡萝卜，**继续炖 20-25 分钟**至土豆绵软
- 加盐调味，大火收汁 3-5 分钟至汤汁浓稠

### 3. 装盘

- 盛入深盘或砂锅中
- 撒上香菜叶或葱花点缀

## 技巧与窍门

- 牛肉要选牛腩（肥瘦相间部位），纯瘦肉炖出来口感发柴
- 土豆不要和牛肉同时下锅，否则炖到牛肉烂时土豆已化成泥
- 炖牛肉全程小火慢炖，大火会让肉质变紧
- 炖煮过程中不要频繁开盖搅拌，保持锅内温度稳定
- 如用高压锅可大大缩短时间：牛肉压 25 分钟后放土豆再压 5 分钟

## 变体

- **番茄炖牛肉**：增加 2 个番茄（切块先炒出汁），酸甜开胃
- **咖喱土豆牛肉**：省去酱油类调料，加咖喱块，做成咖喱风味
- **红酒炖牛肉**：加 200ml 干红葡萄酒代替部分水，西式做法
- **土豆炖牛筋**：牛肉换牛筋，炖制时间延长至 2 小时
