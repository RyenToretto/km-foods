# 麻婆豆腐

> 麻辣鲜香、豆腐嫩滑入味的经典川菜，米饭杀手

## 基本信息

| 项目 | 内容 |
|------|------|
| 分类 | 家常菜 |
| 口味 | 麻辣 |
| 难度 | 中等 |
| 用时 | 约 20 分钟 |
| 份量 | 2-3 人份 |

## 食材

| 食材 | 用量 | 备注 |
|------|------|------|
| 嫩豆腐 | 1 盒（约 400g） | 内酯豆腐或嫩豆腐均可 |
| 猪肉末 | 100g | 三分肥七分瘦，牛肉末也可 |
| 蒜苗 | 2 根 | 切成小粒，最后撒入 |

## 调料

| 调料 | 用量 | 备注 |
|------|------|------|
| 郫县豆瓣酱 | 1.5 大勺（约 25g） | 剁细，正宗麻婆豆腐的灵魂 |
| 花椒粉 | 1 小勺（约 3g） | 最后撒，制造麻味 |
| 干辣椒粉 | 1 小勺 | 可选，增加辣度 |
| 豆豉 | 1 小勺（约 5g） | 剁碎，提鲜 |
| 生抽 | 1 大勺（15ml） | 调味 |
| 白糖 | 1/2 小勺 | 提鲜平衡辣味 |
| 淀粉 | 1 大勺 | 加 4 大勺水调成水淀粉 |
| 食用油 | 2 大勺 | 炒菜用 |
| 盐 | 1/2 小勺 | 焯豆腐用 |
| 热水/高汤 | 200ml | 煮豆腐用 |

## 用具

- 炒锅
- 锅铲（用锅铲轻推，不宜大力翻炒）
- 小煮锅（焯豆腐用）
- 菜刀 + 砧板

## 做法

### 1. 准备工作

- 豆腐切成 2cm 见方的小块
- 烧一锅水，加入 1/2 小勺盐，水开后放入豆腐块，小火煮 2 分钟，轻轻捞出沥干（焯水使豆腐更挺实不易碎）
- 郫县豆瓣酱剁细，豆豉剁碎
- 蒜苗切小粒，淀粉加水调匀备用

### 2. 烹饪步骤

- 锅中倒入食用油，中火放入猪肉末，煸炒至肉末变色出油、微微焦香（约 2 分钟）
- 转小火，放入郫县豆瓣酱和豆豉，炒出红油（约 1 分钟），能闻到浓郁酱香
- 加入干辣椒粉翻炒几下，倒入热水 200ml，加生抽和白糖，搅匀煮开
- 轻轻放入焯好的豆腐块，用锅铲轻推（不要大力翻炒，豆腐会碎），中小火煮 3-4 分钟让豆腐入味
- 将水淀粉再次搅匀，分两次淋入锅中，每次淋入后轻轻晃动锅子使芡汁均匀裹住豆腐，汤汁变浓稠即可

### 3. 装盘

- 盛入深盘中，撒上蒜苗粒和花椒粉，花椒粉要均匀撒满表面

## 技巧与窍门

- 豆腐提前焯盐水是关键步骤，既能去除豆腥味，又能使豆腐更紧实不易碎
- 郫县豆瓣酱要小火炒出红油，这是麻婆豆腐色泽和风味的基础
- 勾芡分两次，第一次让汤汁初步变稠，第二次收到浓亮包裹的状态，行话叫"亮油"
- 花椒粉最后撒而不是炒入，麻味更鲜明突出
- 全程不要大力翻炒豆腐，用锅铲轻推或晃锅即可

## 变体

- **肉末替换**：用牛肉末替代猪肉末，风味更醇厚，更接近正宗做法
- **素食版**：去掉肉末，多加一勺豆瓣酱和少许香菇丁提鲜
- **减辣版**：减少豆瓣酱用量至 1 勺，不放干辣椒粉，保留花椒粉的麻味
