# 寿司卷

> 家庭版手卷寿司，醋饭清爽，食材丰富，自己动手乐趣无穷

## 基本信息

| 项目 | 内容 |
|------|------|
| 分类 | 日韩料理 |
| 口味 | 酸甜清爽 |
| 难度 | 中等 |
| 用时 | 约 50 分钟 |
| 份量 | 3-4 人份（约 6-8 条寿司卷） |

|------|------|------|

| 海苔片（寿司用） | 6-8 张 | 全型海苔，约 20×18cm |

| 蟹肉棒 | 4 根（约 60g） | 撕成条 |
| 鸡蛋 | 2 个 | 做厚蛋烧/蛋皮用 |

## 用具

- 电饭锅
- 寿司竹帘（卷帘）
- 保鲜膜（包裹竹帘防粘）
- 木质或塑料饭盆（拌醋饭用）
- 扇子（给醋饭降温）

## 做法

### 1. 准备工作

- **煮饭**：寿司米淘洗 3 遍至水清澈，浸泡 30 分钟后沥干。米水比 1:1（300g 米加 300ml 水），电饭锅煮熟
- **调寿司醋**：米醋、白糖、盐放入小碗，微波 20 秒或小火加热搅拌至糖盐溶解，放凉
- **拌醋饭**：米饭煮好趁热倒入饭盆，淋入寿司醋，用饭勺以切拌的方式（不要搅拌压碎米粒）翻拌均匀，同时用扇子扇凉至体温温度，盖湿布保温
- **煎蛋烧**：鸡蛋加糖、盐、味醂打匀，平底锅小火分 3 次倒入蛋液，每次凝固后卷起推到一端，再倒下一层。煎好后切成 1cm 宽长条
- **切食材**：三文鱼切 1cm×1cm 长条，黄瓜去籽切长条，牛油果切薄片，蟹肉棒撕条

### 2. 烹饪步骤

- 竹帘上铺一层保鲜膜（防止饭粒粘在竹帘上）
- 取一张海苔，**粗糙面朝上**放在竹帘上（粗糙面更容易粘住米饭）
- 手指蘸水（防粘），取约 150g 醋饭均匀铺在海苔上，**上端留出 2cm 不铺饭**（封口用）
- 在米饭中间偏下位置横向放上食材：三文鱼条 + 牛油果片 + 黄瓜条（或自由搭配）
- **卷制**：双手拇指托起竹帘近端，其余手指固定食材，将竹帘向前卷起，先卷到食材被完全包住，稍微用力压紧定型
- 继续向前滚动至海苔卷完全封口，利用留出的海苔边缘粘合（可在边缘抹少许水帮助粘合）
- 用竹帘包裹整条寿司卷，轻轻按压定型成圆形或方形
- 重复以上步骤完成所有寿司卷

### 3. 装盘

- 刀刃蘸水（每切一刀蘸一次，防止粘饭），将寿司卷切成 **2-3cm 厚的段**，共约 6-8 片
- 寿司段立起摆放在盘中
- 在部分寿司上撒白芝麻
- 旁边放上寿司姜片，小碟中倒酱油和芥末

## 技巧与窍门

- 醋饭是寿司的灵魂，切拌时要轻柔避免压碎米粒，同时扇风让醋饭快速降温表面产生光泽
- 醋饭的温度应接近体温（35-40°C），太热会让海苔变软，太冷饭粒硬不好吃
- 铺饭不要铺太厚，约 5mm 即可，太厚卷出来咬不动
- 卷的时候第一下最关键：要把食材完全兜住，然后再整体滚动
- 切寿司时刀一定要蘸水且要锋利，钝刀会把寿司压扁
- 刺身级三文鱼购买后冷藏保存，当天食用

## 变体

- **反转卷（里卷）**：米饭在外海苔在内，外层裹白芝麻或鱼籽
- **手握寿司（Nigiri）**：手捏成饭团放上鱼片，更考验手法
- **手卷（Temaki）**：海苔对折成甜筒形，装入饭和食材，适合聚餐自助
- **素寿司**：腌萝卜条 + 黄瓜 + 牛油果 + 腐皮，纯素也很美味
- **炸虾天妇罗卷**：内馅换成炸虾天妇罗，外面淋上照烧酱
