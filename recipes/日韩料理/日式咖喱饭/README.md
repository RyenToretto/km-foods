# 日式咖喱饭

> 浓郁微甜的日式咖喱，搭配软糯米饭，令人欲罢不能的治愈系美食

## 基本信息

| 项目 | 内容 |
|------|------|
| 分类 | 日韩料理 |
| 口味 | 甜咸微辛 |
| 难度 | 简单 |
| 用时 | 约 50 分钟 |
| 份量 | 3-4 人份 |

## 食材

| 食材 | 用量 | 备注 |
|------|------|------|
| 鸡腿肉/猪肉片 | 300g | 鸡腿肉切 3cm 块，猪肉切厚片 |
| 土豆 | 2 个（约 300g） | 去皮切滚刀块 |
| 胡萝卜 | 1 根（约 150g） | 切滚刀块 |
| 洋葱 | 1 个（约 200g） | 切粗丝 |
| 米饭 | 4 碗 | 煮好的白米饭 |
| 苹果 | 半个（约 80g） | 磨泥，增甜提鲜，可选 |

## 调料

| 调料 | 用量 | 备注 |
|------|------|------|
| 日式咖喱块 | 半盒（约 100g） | 好侍/House 黄金咖喱中辣推荐 |
| 食用油 | 2 汤匙（约 30ml） | — |
| 水 | 700ml | — |
| 酱油 | 1 茶匙（约 5ml） | 提鲜 |
| 蜂蜜 | 1 茶匙（约 7g） | 增加浓郁感，可选 |
| 盐 | 少许 | 最后调味 |

## 用具

- 汤锅或厚底深锅
- 木铲

## 做法

### 1. 准备工作

- 鸡腿肉去骨切 3cm 块（保留鸡皮更香）
- 土豆、胡萝卜去皮切滚刀块（大小约 3cm，比平时切大一号，久煮不容易化）
- 洋葱切 1cm 粗丝
- 苹果磨泥备用

### 2. 烹饪步骤

- 锅中加油，**中火**先炒洋葱丝至变软微黄（约 5 分钟），洋葱炒透是日式咖喱甜味的来源
- 加入鸡肉块翻炒至表面变白，约 3 分钟
- 加入土豆和胡萝卜翻炒 2 分钟，使表面裹油
- 加入 700ml 水，大火烧开后撇去浮沫
- **转中小火，盖盖煮 20 分钟**至土豆和胡萝卜可以用筷子轻松插入
- **关火**，将咖喱块掰成小块放入锅中，搅拌至完全融化（关火加咖喱块可避免结块糊底）
- 加入苹果泥、酱油和蜂蜜
- **再次开小火，煮 5-8 分钟**，不断搅拌至酱汁浓稠
- 尝味后根据需要加盐调整

### 3. 装盘

- 米饭盛在盘子一侧（可用碗倒扣塑形）
- 咖喱浇在另一侧，让咖喱和米饭有部分交汇
- 可放上福神渍（红色腌菜）或煎蛋作为点缀

## 技巧与窍门

- 咖喱块一定要关火后再加，开火状态下加容易结块并粘锅底
- 洋葱要炒到位，炒得越透越甜，是日式咖喱区别于其他咖喱的关键
- 隔夜咖喱比新做的更好吃，因为土豆的淀粉会让酱汁更浓稠
- 不同品牌咖喱块的辣度和风味差异很大，可以混合两种品牌/辣度
- 加入苹果泥和蜂蜜是日式家庭咖喱的秘方，使酱汁更加圆润浓郁

## 变体

- **炸猪排咖喱饭**：配一块炸猪排（Katsu Curry），是最受欢迎的日式咖喱吃法
- **海鲜咖喱**：鸡肉换成虾仁和鱿鱼，最后 5 分钟加入
- **咖喱乌冬**：米饭换成乌冬面，多加水使酱汁更稀，做成汤咖喱
- **烤蔬菜咖喱**：纯素版，用烤茄子、烤南瓜、烤西葫芦搭配
