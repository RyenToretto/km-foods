# 泰式绿咖喱鸡

> 椰香浓郁的泰式经典咖喱，绿咖喱酱与椰浆的完美结合，微辣中带着清新草本香

## 基本信息

| 项目 | 内容 |
|------|------|
| 分类 | 东南亚 |
| 口味 | 椰香微辣 |
| 难度 | 中等 |
| 用时 | 约 35 分钟 |
| 份量 | 3-4 人份 |

## 食材

| 食材 | 用量 | 备注 |
|------|------|------|
| 鸡腿肉 | 400g | 去骨切 2cm 块 |
| 泰国茄子 | 2 个（约 150g） | 可用普通圆茄切块替代 |
| 青椒 | 1 个 | 切菱形块 |
| 红椒 | 半个 | 切菱形块，增色用 |
| 竹笋 | 100g | 切薄片 |
| 九层塔（泰国罗勒） | 一把（约 20g） | 最后放，不可省略 |
| 青柠叶 | 4 片 | 撕碎 |

## 调料

| 调料 | 用量 | 备注 |
|------|------|------|
| 绿咖喱酱 | 3 大勺（约 60g） | 推荐 Mae Ploy 品牌 |
| 椰浆 | 400ml（1 罐） | 浓椰浆，开罐前不要摇 |
| 鱼露 | 2 大勺（约 30ml） | 提鲜调咸 |
| 椰糖 | 1 大勺（约 15g） | 可用红糖替代 |
| 食用油 | 1 大勺（约 15ml） | 炒咖喱酱用 |

## 用具

- 炒锅或深平底锅
- 砧板与刀
- 汤勺

## 做法

### 1. 准备工作

- 鸡腿肉去骨去皮，切成 2cm 见方的块
- 泰国茄子切滚刀块，竹笋切薄片
- 青红椒切菱形块，九层塔洗净摘叶
- 椰浆罐头不摇晃开盖，勺出上层浓稠部分（约 100ml）备用，剩余为稀椰浆

### 2. 烹饪步骤

- 锅中倒入食用油，中火加热，舀入上层浓椰浆，持续搅拌 2-3 分钟至椰油析出、表面微微冒泡
- 加入绿咖喱酱，翻炒 1-2 分钟至香气浓郁、酱料颜色变深
- 放入鸡肉块，翻炒 3-4 分钟至表面变白裹上咖喱酱
- 倒入剩余稀椰浆，放入青柠叶，大火煮沸后转中小火
- 加入茄子和竹笋，煮 8-10 分钟至鸡肉熟透、茄子变软
- 放入青红椒块，煮 2 分钟保持脆感
- 加鱼露和椰糖调味，关火后放入九层塔叶拌匀

### 3. 装盘

- 盛入深盘或碗中，表面点缀几片九层塔叶和红椒丝
- 配泰国茉莉香米饭，咖喱汁拌饭极佳

## 技巧与窍门

- 浓椰浆先炒出油脂再下咖喱酱，是泰式咖喱风味浓郁的关键步骤
- 九层塔一定要最后放、关火后拌入，久煮会发黑变苦
- 绿咖喱酱本身含盐，鱼露要少量多次添加避免过咸
- 鸡腿肉比鸡胸肉更适合炖煮，口感嫩滑不柴
- 椰糖的作用是平衡辣味和咸味，不可省略

## 变体

- **绿咖喱虾**：鸡肉换成大虾，最后 3 分钟再放入，避免虾肉过老
- **绿咖喱牛肉**：牛腩切块，炖煮时间延长至 40-50 分钟
- **素食版**：鸡肉换成油豆腐和杏鲍菇，鱼露换成酱油
