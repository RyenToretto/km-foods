# 越南春卷

> 清爽透明的鲜虾春卷，米皮裹着缤纷蔬菜与弹嫩虾仁，蘸上花生酱料一口入魂

## 基本信息

| 项目 | 内容 |
|------|------|
| 分类 | 东南亚 |
| 口味 | 清爽鲜甜 |
| 难度 | 简单 |
| 用时 | 约 30 分钟 |
| 份量 | 2-3 人份（约 10 个） |

| 越南米皮 | 10 张 | 22cm 圆形，超市进口区有售 |
| 鲜虾 | 200g（约 10 只） | 中号虾，去壳去虾线 |
| 米粉（细） | 50g | 干米粉 |
| 生菜 | 5 片 | 洗净沥干 |

| 薄荷叶 | 一小把（约 10g） | 提香关键 |
| 香菜 | 一小把（约 10g） | 可选 |

## 用具

- 大盘或砧板（卷春卷用）
- 大碗（泡米皮用）
- 小锅（煮虾和米粉）

## 做法

### 1. 准备工作

- 鲜虾煮熟（水沸后放入，煮 2 分钟至虾身弯曲变红），捞出过冰水，沿背部对半片开
- 米粉用沸水煮 3-4 分钟至透明柔软，捞出过冰水沥干，剪成 10cm 段
- 胡萝卜、黄瓜、紫甘蓝切成 6-7cm 长的细丝
- 生菜叶撕成适合包裹的大小
- 调蘸料：花生酱 + 海鲜酱 + 鱼露 + 青柠汁 + 蒜末 + 小米辣碎，加温水搅拌至顺滑浓稠

### 2. 烹饪步骤

- 准备一盆温水（约 40°C，手感温热不烫），将一张米皮放入浸泡 5-8 秒，至米皮变软但仍有韧性
- 取出铺在湿润的砧板或大盘上
- 在米皮下半部铺一片生菜叶，依次摆上米粉、胡萝卜丝、黄瓜丝、紫甘蓝丝
- 放上薄荷叶和香菜
- 先将米皮下端向上折盖住馅料，再将左右两侧向内折
- 在上半部分放 2-3 片对半切的虾仁（红色面朝下，卷好后透过米皮可以看到虾的颜色）
- 继续向上卷紧，收口朝下放置
- 重复以上步骤完成所有春卷

### 3. 装盘

- 将春卷整齐排列在盘中，可对半斜切展示内部色彩
- 蘸料倒入小碗，撒上碎花生，随春卷一起上桌

## 技巧与窍门

- 米皮不要泡太久，5-8 秒足够，过软容易破裂粘连
- 卷的时候手指保持湿润，防止粘手
- 馅料不要放太多，每种少量即可，否则卷不紧容易散
- 虾仁红色面朝外放在最后一圈，成品透过米皮看到虾肉更美观
- 卷好的春卷用湿布盖住防干裂，尽快食用口感最佳
- 温水泡米皮比热水好控制，不容易泡过头

## 变体

- **鸡丝版**：虾换成手撕白切鸡丝，更有饱腹感
- **素食版**：去掉虾，加牛油果片和炸豆腐条
- **芒果版**：加入芒果条，搭配甜辣酱，酸甜清爽
