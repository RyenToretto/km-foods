# 冬阴功汤

> 泰式经典酸辣虾汤，香茅与青柠交织的热带风味，一口就让人上瘾

## 基本信息

| 项目 | 内容 |
|------|------|
| 分类 | 东南亚 |
| 口味 | 酸辣鲜香 |
| 难度 | 中等 |
| 用时 | 约 40 分钟 |
| 份量 | 3-4 人份 |

## 食材

| 食材 | 用量 | 备注 |
|------|------|------|
| 鲜虾 | 300g（约 12 只） | 中等大小，带壳 |
| 草菇 | 150g | 可用蘑菇替代 |
| 小番茄 | 6 个 | 对半切开 |
| 香茅 | 3 根 | 取白色部分，拍扁切段 |
| 南姜 | 30g | 切薄片，不可用生姜替代 |
| 青柠叶 | 6 片 | 撕去中间硬梗 |
| 小米辣 | 4-6 个 | 拍扁，根据耐辣程度调整 |
| 青柠 | 2 个 | 取汁 |

## 调料

| 调料 | 用量 | 备注 |
|------|------|------|
| 冬阴功酱 | 2 大勺（约 40g） | 超市有售，品牌推荐佳乐 |
| 椰浆 | 200ml | 浓椰浆效果更好 |
| 鱼露 | 2 大勺（约 30ml） | 提鲜关键 |
| 白砂糖 | 1 小勺（约 5g） | 平衡酸味 |
| 食用油 | 1 大勺（约 15ml） | 炒酱用 |

## 用具

- 汤锅（3L 以上）
- 砧板与刀
- 汤勺

## 做法

### 1. 准备工作

- 鲜虾去虾线，剥壳留尾，虾壳和虾头留用（煮高汤提鲜）
- 草菇洗净对半切开，小番茄对半切开
- 香茅拍扁切成 3cm 段，南姜切 2mm 薄片，青柠叶撕去中梗
- 小米辣拍扁，青柠切半备用

### 2. 烹饪步骤

- 锅中倒入食用油，中火加热至微微冒烟，放入虾头虾壳翻炒 2-3 分钟至变红出油
- 加入 800ml 清水，大火煮沸后转小火熬 10 分钟，滤出虾壳得虾汤底
- 虾汤重新倒入锅中，大火煮沸，放入香茅、南姜、青柠叶煮 5 分钟出香
- 加入冬阴功酱搅匀，放入草菇煮 3 分钟至变软
- 放入鲜虾和小番茄，中火煮 2-3 分钟至虾身弯曲变红
- 倒入椰浆搅匀，加鱼露和白砂糖调味
- 关火后挤入青柠汁，放入拍扁的小米辣

### 3. 装盘

- 盛入深碗，表面可点缀几片香菜叶和薄荷叶
- 搭配泰国香米饭更佳

## 技巧与窍门

- 青柠汁一定要关火后再加，高温会让青柠发苦
- 虾头虾壳炒出红油再熬汤，鲜味比直接用清水强很多
- 南姜和香茅是灵魂配料，不建议省略或替代
- 椰浆分量可根据个人喜好增减，不加椰浆就是清汤版（泰语叫 Tom Yum Nam Sai）
- 鱼露要分次加，边加边尝，避免过咸

## 变体

- **清汤版**：不加椰浆，酸辣味更突出，适合怕腻的人
- **海鲜版**：加入鱿鱼圈、蛤蜊、青口贝，海鲜味更丰富
- **素食版**：去掉虾，换成豆腐和杏鲍菇，鱼露换成酱油
