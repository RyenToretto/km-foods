# 葱油拌面

> 上海风味经典拌面，焦香葱油裹住每根面条，简单却令人上瘾

## 基本信息

| 项目 | 内容 |
|------|------|
| 分类 | 主食 |
| 口味 | 咸鲜微甜 |
| 难度 | 简单 |
| 用时 | 约 20 分钟 |
| 份量 | 2 人份 |

## 食材

| 食材 | 用量 | 备注 |
|------|------|------|
| 细挂面或新鲜碱面 | 200 克 | 碱面口感更好 |
| 小葱 | 8-10 根（约 80 克） | 这道菜的灵魂，不能少 |
| 洋葱 | 1/4 个（约 50 克） | 可选，增加甜度 |

## 调料

| 调料 | 用量 | 备注 |
|------|------|------|
| 食用油 | 80 毫升 | 色拉油或花生油 |
| 生抽 | 3 汤匙（45 毫升） | — |
| 老抽 | 1 汤匙（15 毫升） | 上色用 |
| 白糖 | 1.5 汤匙（20 克） | 上海风味偏甜 |
| 开水 | 2 汤匙（30 毫升） | 化糖用 |

## 用具

- 小锅或平底锅（熬葱油）
- 煮面锅
- 漏勺
- 拌面碗

## 做法

### 1. 准备工作

- 小葱洗净沥干水分（水分会导致油溅），切成 5 厘米长的段
- 洋葱切丝
- 生抽、老抽、白糖和开水混合搅拌至糖溶化，调成酱汁备用

### 2. 熬葱油

- 小锅或平底锅中倒入食用油，冷油下入葱段和洋葱丝
- 开小火，慢慢加热，让葱在油中缓慢脱水
- 保持小火慢炸约 8-10 分钟，期间偶尔翻动，让葱受热均匀
- 当葱段变成深棕色、边缘发脆时关火（颜色变化很快，注意别炸糊）
- 趁油温高时将调好的酱汁倒入葱油中，会剧烈沸腾，搅匀即成葱油酱汁

### 3. 煮面与装盘

- 另起一锅烧水至沸腾，下入面条
- 挂面煮约 3 分钟，碱面煮约 2 分钟，尝一下确认熟透但仍有弹性
- 捞出面条沥水，放入碗中
- 浇上 2-3 汤匙葱油酱汁（连葱带油），充分拌匀
- 表面再放几根炸好的焦葱装饰

## 技巧与窍门

- 熬葱油全程小火是关键，大火会让葱外焦内生，香味出不来
- 葱必须完全沥干水分再下锅，否则会溅油
- 加洋葱一起炸可以增加葱油的甜香层次感
- 酱汁倒入热油时要注意安全，油温高会溅起
- 葱油酱汁可以一次多做，放入密封玻璃瓶冷藏保存 2 周
- 面条捞出后不要过冷水，否则酱汁不容易挂在面上

## 变体

- **猪油版**：用 40 克猪油加 40 毫升植物油混合来熬，香味更浓郁
- **加料版**：拌好的面上铺一个煎蛋和几片卤牛肉，更有满足感
- **拌馄饨**：同样的葱油酱汁浇在煮好的小馄饨上，就是上海葱油馄饨
