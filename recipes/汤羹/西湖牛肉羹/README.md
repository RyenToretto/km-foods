# 西湖牛肉羹

> 杭帮名汤，牛肉细嫩蛋花如丝，口感滑润鲜香，宴客家常皆宜

## 基本信息

| 项目 | 内容 |
|------|------|
| 分类 | 汤羹 |
| 口味 | 咸鲜 |
| 难度 | 中等 |
| 用时 | 约 30 分钟 |
| 份量 | 3-4 人份 |

## 食材

| 食材 | 用量 | 备注 |
|------|------|------|
| 牛里脊肉 | 150g | 逆纹理切薄片后剁成细末 |
| 鸡蛋 | 3 个 | 只取蛋清 |
| 豆腐 | 100g（约 1/4 盒） | 切 0.5cm 小丁 |
| 香菇 | 3 朵（约 30g） | 鲜香菇或泡发干香菇，切细丁 |
| 香菜 | 2 根 | 切碎，装盘用 |

## 调料

| 调料 | 用量 | 备注 |
|------|------|------|
| 生抽 | 1 大勺（15ml） | 腌牛肉用 |
| 料酒 | 1 大勺（15ml） | 腌牛肉去腥 |
| 淀粉 | 1 小勺 | 腌牛肉用 |
| 盐 | 1 小勺（约 5g） | 调汤底 |
| 白胡椒粉 | 1/2 小勺 | 提味去腥，西湖牛肉羹的标志性调味 |
| 香醋 | 1 小勺（5ml） | 出锅前淋入，提鲜 |
| 香油 | 1 小勺（5ml） | 出锅时淋入 |
| 淀粉 | 2 大勺 | 加 6 大勺水调成水淀粉，勾芡用 |
| 鸡汤/清水 | 800ml | 用鸡汤风味更佳 |

## 用具

- 汤锅
- 打蛋碗
- 漏勺或筷子（淋蛋清用）

## 做法

### 1. 准备工作

- 牛里脊逆纹理切薄片，再剁成细末（不要用绞肉机，手剁的口感更好）
- 牛肉末加生抽 1 大勺、料酒 1 大勺、淀粉 1 小勺，抓匀腌制 10 分钟
- 鸡蛋分离蛋清蛋黄，蛋清用筷子轻轻打散（不要打出泡沫），蛋黄另作他用
- 豆腐切 0.5cm 小丁，香菇切细丁
- 水淀粉调匀备用

### 2. 烹饪步骤

- 锅中倒入鸡汤（或清水）800ml，大火烧开
- 放入香菇丁，煮 2 分钟出香味
- 放入腌好的牛肉末，用筷子快速搅散（避免结团），煮至牛肉变色（约 1 分钟），撇去浮沫
- 放入豆腐丁，加盐和白胡椒粉调味，煮 1 分钟
- 将水淀粉再次搅匀，缓缓淋入锅中，同时用勺子轻轻搅动，使汤羹变得浓稠（浓度如米汤，勺子舀起能缓缓流下）
- 转最小火，将蛋清沿筷子细细淋入锅中，边淋边用勺子轻轻推动，让蛋清形成细丝状飘散
- 关火，淋入香醋和香油

### 3. 装盘

- 盛入汤碗中，撒上香菜碎，可再磨少许白胡椒粉在表面

## 技巧与窍门

- 牛肉必须手剁成末，纹理保留让口感更好；绞肉机打的太细会变成肉泥口感
- 蛋清不要打发起泡，轻轻搅散即可，否则倒入汤中会出现泡沫影响观感
- 勾芡是这道汤的关键，浓度要把握好——太稀没有"羹"的感觉，太稠则口感厚重
- 淋蛋清时火一定要小，汤不能沸腾，否则蛋清会变成碎块而不是飘逸的丝状
- 白胡椒粉是这道汤的灵魂调味，不可省略

## 变体

- **蟹肉版**：加入 50g 蟹肉同煮，升级为蟹肉牛肉羹
- **素食版**：去掉牛肉，用 100g 口蘑切丁替代，其余步骤不变
- **浓汤版**：增加水淀粉用量至 3 大勺，做成更浓稠的宴席版
