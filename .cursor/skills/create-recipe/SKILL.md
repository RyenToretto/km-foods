# 创建新菜谱

## 工作流程

### 第 1 步：收集信息

使用 AskQuestion 工具（如可用）或对话方式，向用户收集：

1. **菜名** — 必填
2. **分类** — 从分类列表中选择，见 [reference.md](reference.md)
3. **口味** — 从口味列表中选择，见 [reference.md](reference.md)
4. **难度** — 简单 / 中等 / 困难
5. **大致用时**
6. **份量**
7. **食材与用量**
8. **调料与用量**
9. **所需用具**
10. **做法步骤**
11. **技巧窍门**（可选）
12. **变体做法**（可选）

用户可以一次性给出所有信息，也可以只给菜名，由 AI 根据烹饪知识补全后请用户确认。

### 第 2 步：确定路径

文件路径为 `recipes/{分类}/{菜名}/README.md`。如果分类目录不存在则创建。

### 第 3 步：生成菜谱

读取 `templates/recipe-template.md` 模板，用收集到的信息填充，生成完整菜谱。

格式要求：
- 食材和调料的用量尽量精确（克、毫升、勺、个）
- 做法步骤包含火候、时间、感官判断标准
- 技巧来自实际烹饪经验

### 第 4 步：确认

将生成的菜谱展示给用户确认，根据反馈修改后写入文件。

- 格式模板见 `templates/recipe-template.md`
