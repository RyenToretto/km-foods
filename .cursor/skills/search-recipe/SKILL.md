# 搜索菜谱

## 指令

在 `recipes/` 目录中搜索菜谱，支持多种筛选条件。

### 第 1 步：扫描菜谱库

使用 Glob 工具查找所有 `recipes/**/README.md` 文件，读取每份菜谱的基本信息部分。

### 第 2 步：根据条件筛选

支持的筛选维度：

| 条件 | 示例 |
|------|------|
| 食材 | "有鸡蛋和番茄能做什么" |
| 分类 | "有什么汤羹" |
| 口味 | "想吃酸甜的" |
| 难度 | "有什么简单的菜" |
| 用时 | "20 分钟以内能做什么" |
| 关键词 | "和豆腐有关的菜" |

### 第 3 步：输出结果

以表格格式列出匹配的菜谱：

```markdown
| 菜名 | 分类 | 口味 | 难度 | 用时 |
|------|------|------|------|------|
| ... | ... | ... | ... | ... |
```

如果没有匹配结果，告知用户并建议放宽条件。

如果用户根据已有食材搜索但没有完全匹配的菜谱，可以推荐最接近的菜谱并说明缺少哪些食材。
