# 分析菜谱

## 指令

对指定菜谱进行多维度分析，输出结构化的分析报告。

### 第 1 步：读取菜谱

读取目标菜谱的 `README.md`，提取食材、调料、用量和做法信息。

### 第 2 步：营养估算

根据食材和用量，估算每份的：
- 总热量（千卡）
- 蛋白质（克）
- 脂肪（克）
- 碳水化合物（克）
- 膳食纤维（克）
- 钠含量（毫克）

注明：估算基于常见食材营养数据，仅供参考。

### 第 3 步：健康评估

- 整体健康度评级：优 / 良 / 中 / 差
- 亮点：如高蛋白、富含维生素等
- 注意点：如高盐、高油、高糖等

### 第 4 步：烹饪分析

- 难度是否标注准确
- 用时是否合理
- 步骤是否清晰完整

### 第 5 步：输出报告

以 Markdown 格式输出分析报告，包含上述所有维度。如果发现问题，给出改进建议。
